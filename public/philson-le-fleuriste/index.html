How booking works

The public lane stays simple: request first, review second, deposit next, execution after approval.

1

Start Request

Start the deposit request and send the date, city, project type, budget range, and floral direction.

2

Review + Confirm

Philson reviews timing, availability, floral scope, and whether the request fits the production calendar.

3

Deposit Stage

Once approved, the deposit and booking terms are confirmed so the project can move into the locked calendar lane.

4

Build + Deliver

After booking is confirmed, the floral work moves into build, finish, pickup, delivery, or on-site setup.
